Images to be displayed as logos in the application should be a minimum of 202x135 pixels. Larger images will be resized to fit. Supported image types are .jpg, .gif, .png.

Only CSV File format is accepted. The individual fields must be structured in the order shown in the sample below and must be separated by commas. A header row is also required.

First Name,Last Name,Email Address,Business Phone,Home Phone,Mobile Phone Paul,Hill,paul.hill@abc.com,416-123-4567x423,905-765-4321,416-231-4567

For more details on how to format your file, or how to export your personal address book, please click on the Help link above.
